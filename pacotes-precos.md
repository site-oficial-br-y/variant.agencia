# Pacotes e Preços — Variant (uso interno)

> **Regra fixa:** TODOS os pacotes já incluem **todas as formas de contato**
> (WhatsApp, telefone, e-mail, Instagram e localização no Maps).
> Contato é padrão — nunca é diferencial de preço.
>
> O que muda o preço é: **nº de páginas/seções + funcionalidade**.

---

## Tabela de pacotes

| Pacote | O que inclui (além de todos os contatos) | Preço |
|---|---|---|
| **Básico** (1 página) | História da empresa, equipe, Maps, todos os contatos, design responsivo | **R$250** |
| **Profissional** | + serviços detalhados, galeria de fotos, depoimentos, várias seções | **R$450–600** |
| **Completo** | + formulário de contato, integração com Instagram, design caprichado | **R$700–900** |
| **Com Sistema** | Site + 1 funcionalidade real (ver abaixo) | **R$999–1.500** |
| **Sistema Sob Medida** | Plataforma com várias funcionalidades (tipo a Honk) | **R$2.000–5.000+** |

---

## O que é "Com Sistema" (R$999–1.500)

Entra backend, banco de dados (Supabase), login, pagamento — software de verdade:

- 📅 Agendamento online (clínica, salão, barbearia) → ~R$1.200
- 🍔 Cardápio digital com pedido (restaurante)
- 🔐 Área de login / portal do cliente
- 💳 Integração de pagamento (Pix / Mercado Pago)
- 📊 Painel pro dono editar o próprio site

## Sistema Sob Medida (R$2.000–5.000+)

Plataforma completa — várias funções juntas:
- Loja virtual / e-commerce
- Sistema de assinatura/mensalidade (igual a Honk)
- CRM, dashboard, multiusuário

---

## Pagamento

- **R$100 de sinal** pra começar (já dentro do valor total).
- Restante **só quando o site estiver pronto e aprovado**.
- Site só vai ao ar **depois do pagamento** (a chave fica com você até lá).

## Como encaixar o cliente no pacote

Pergunta-chave: *"você quer só mostrar a empresa, ou quer que o site **faça** algo — agendar, vender, receber pagamento?"*

- Só apresentar → **Básico / Profissional**
- Mostrar muito (serviços, fotos, depoimentos) → **Profissional / Completo**
- Quer que o site **faça** algo → **Com Sistema** (R$999+)
