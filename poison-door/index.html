⚙️ COMO CONFIGURAR ESTE SITE

1. Número de WhatsApp da banda
Abra index.html, localize CONFIG.whatsapp e substitua 'SEU_NUMERO_AQUI' pelo número com código do país, sem espaços ou traços.
Exemplo: whatsapp: '5511999887766'

2. Google Calendar (mostrar datas ocupadas)

  1. Crie um Google Calendar chamado "Poison Door – Shows" (ou use um existente)
  2. Vá em Configurações → marque Tornar público
  3. Copie o Calendar ID (parece com abc123@group.calendar.google.com)
  4. No Google Cloud Console, crie um projeto, habilite a Google Calendar API e gere uma API Key restrita ao seu domínio GitHub Pages
  5. Cole os valores em CONFIG.googleCalendarId e CONFIG.googleApiKey
  6. Adicione eventos no Google Calendar — o site bloqueia as datas automaticamente

3. Hospedar no GitHub Pages
Crie um repositório no GitHub, coloque este arquivo como index.html na raiz, depois vá em Settings → Pages → Source: main branch / (root) e salve.
O site ficará disponível em https://SEU-USUARIO.github.io/NOME-DO-REPO
